# Source Policy

## Preferred Sources

1. official product or library docs
2. research papers or specifications
3. official vendor blogs or release notes
4. reputable secondary sources only when primary material is unavailable

## Use Context7 For

- official Codex docs
- library/framework docs in Context7
- version-sensitive tool behavior

## Output Expectations

- identify the claim
- identify the source used
- state whether the claim is confirmed, needs nuance, or should be removed
